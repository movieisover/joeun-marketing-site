# 조은마케팅 브랜드 가이드라인

> Brand Guidelines v1.0
> 일관된 브랜드 경험을 위한 기준 문서

이 문서는 조은마케팅의 로고, 색상, 서체, 보이스 톤, 적용 사례를 정의합니다. 외부 제작물·SNS·광고 등 모든 접점에서 본 가이드를 따릅니다.

---

## 1. 브랜드 에센스

| 항목 | 정의 |
|------|------|
| 브랜드 한 줄 | 작은 가게의 진심을, 더 많은 고객에게 |
| 본질(Essence) | 소상공인 곁의 든든한 마케팅 파트너 |
| 핵심 약속 | 비용 대비 확실한 성과와 완전한 투명성 |
| 톤 | 신뢰감 70% + 친근함 30% |

### 브랜드 성격 (Brand Personality)
- **믿음직한 (Reliable)** — 약속한 성과를 숫자로 증명한다
- **친근한 (Approachable)** — 동네 사장님 눈높이로 대화한다
- **명료한 (Clear)** — 어려운 용어 대신 쉬운 말로 설명한다
- **성실한 (Diligent)** — 작은 가게도 끝까지 챙긴다

---

## 2. 브랜드 포지셔닝

> 대기업 마케팅 전략을, 동네 사장님도 쓸 수 있게.

조은마케팅은 "고급스러운 대행사"가 아니라 **"믿고 맡기는 옆집 마케팅 전문가"** 로 인식되는 것을 목표로 합니다. 모든 브랜드 표현은 다음 축 위에서 움직입니다.

```
   전문적
     │
     │   ● 조은마케팅
     │  (전문적이되 따뜻한 영역)
     │
대중적 ─────────────── 고급
     │
     │
   친근함
```

## 3. 로고

### 3-1. 로고 구성
- **심볼 + 워드마크** 조합형을 기본으로 사용
- 심볼 모티프: 성장(상승 그래프) + 진심(말풍선/하트)을 결합한 형태
- 워드마크: "조은마케팅" 한글 전용 / "JOEUN" 영문 보조

### 3-2. 로고 사용 규칙

**여백 (Clear Space)**
- 로고 높이(심볼 기준)의 **0.5배 이상** 여백을 사방에 확보

**최소 크기 (Minimum Size)**
| 매체 | 최소 크기 |
|------|-----------|
| 디지털 | 가로 80px 이상 |
| 인쇄 | 가로 20mm 이상 |

### 3-3. 금지 사항 (Don'ts)
- 로고 비율 임의 변형(가로·세로 늘이기) 금지
- 승인되지 않은 색상으로 변경 금지
- 그림자·외곽선·입체 효과 추가 금지
- 복잡한 배경 위 직접 배치 금지(가독성 확보 시 박스 처리)
- 워드마크 서체 임의 변경 금지

## 5. 타이포그래피

### 5-1. 기본 서체
| 용도 | 서체 | 비고 |
|------|------|------|
| 국문 기본 | Pretendard | 헤드라인·본문 공용 |
| 영문/숫자 | Pretendard | 국문과 동일 패밀리 |
| 대체 서체 | 본고딕 (Noto Sans KR) | Pretendard 미사용 환경 |

### 5-2. 위계 (Type Scale)
| 단계 | 굵기 | 용도 |
|------|------|------|
| H1 | Bold | 메인 타이틀 |
| H2 | SemiBold | 섹션 제목 |
| H3 | SemiBold | 소제목 |
| Body | Regular | 본문 |
| Caption | Regular | 보조 설명·각주 |

### 5-3. 가독성 규칙
- 본문 자간은 기본값 유지, 헤드라인은 살짝 좁게(-2~-3%) 가능
- 행간(line-height)은 본문 기준 1.6 권장
- 한 줄 길이는 모바일 가독성을 위해 너무 길지 않게 조정

## 6. 보이스 & 톤 (Voice & Tone)

### 6-1. 기본 원칙
조은마케팅의 말은 **"전문가가 쉽게 설명해 주는 느낌"** 이어야 합니다.

| 이렇게 (Do) | 이렇게 하지 않기 (Don't) |
|-------------|--------------------------|
| 쉬운 우리말로 풀어 설명 | 전문 용어 남발 |
| 성과를 숫자로 제시 | 막연한 미사여구 |
| 사장님 입장에서 공감 | 일방적 자랑·과장 |
| 정중하되 친근한 존댓말 | 지나치게 딱딱하거나 가벼운 말투 |

### 6-2. 상황별 톤
| 상황 | 톤 |
|------|-----|
| 영업·제안 | 신뢰감 있고 명료하게 |
| SNS·콘텐츠 | 친근하고 가볍게 |
| 리포트·안내 | 정확하고 차분하게 |
| 사과·이슈 대응 | 진솔하고 책임감 있게 |

### 6-3. 메시지 예시
- (O) "이번 달 플레이스 방문 문의가 지난달보다 2배 늘었어요."
- (X) "혁신적 그로스 해킹으로 폭발적 트래픽을 견인했습니다."

## 7. 이미지 · 비주얼 스타일

- **사진**: 실제 가게·사장님·손님이 담긴 현장감 있는 사진 우선
- **분위기**: 밝고 따뜻한 톤, 과한 보정 지양
- **그래픽**: 깔끔한 라인·여백 중심, 정보는 도식으로 단순하게
- **아이콘**: 일관된 라인 스타일, 그린 계열 포인트

> 스톡 느낌의 과장된 비즈니스 이미지보다 **진짜 현장**을 보여주는 것이 브랜드 신뢰와 직결됩니다.

## 8. 적용 사례 (Applications)

| 매체 | 적용 기준 |
|------|-----------|
| 명함 | 화이트 배경 + 그린 로고, 옐로우 포인트 라인 |
| SNS 프로필 | 그린 배경 + 화이트 심볼 |
| 콘텐츠 썸네일 | 화이트/라이트그린 배경 + 잉크 헤드라인 + 옐로우 강조 |
| 제안서·리포트 | 화이트 기반, 그린 헤더, 표 라인은 라인그레이 |
| 간판·현수막 | 그린 바탕 + 화이트 워드마크 (원거리 가독성 우선) |
